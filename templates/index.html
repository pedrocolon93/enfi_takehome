What is RStd?

RStd (Recall Standard Deviation) is the primary metric for measuring selection bias. Rather than simply counting how often the model picks each letter (which would be skewed by label imbalance in the dataset), it measures the balance of per-letter recalls — how well the model correctly identifies answers when the true answer is A, vs. B, vs. C, vs. D, vs. E.

The standard deviation across those recall values gives a single number for bias severity. A perfectly unbiased model would have equal recall for all letters, giving RStd = 0. Higher values indicate the model is systematically better at recognizing correct answers in certain positions than others.

The value shown above is the average RStd across all permutations. Individual per-permutation RStd values are shown in the table below.
