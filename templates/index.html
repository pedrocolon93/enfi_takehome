What is ODS?

The Option Dependency Score (ODS) measures how much a model's answer changes when the correct answer is moved to different positions among the choices.

It works by running the same set of questions with the correct answer placed at each position (A through E), then measuring the variance of how often the model selects each letter across those permutations. If the model always picks the same distribution regardless of where the correct answer is, the variance is zero and ODS = 0.0 (perfectly order-invariant). If the model's selections shift dramatically based on position, variance is high and ODS approaches 1.0 (completely order-dependent).

Formally: for each letter L, we compute the variance of its selection probability across the 5 permutations, then average those variances and normalize by the theoretical maximum (0.16).

  • ODS < 0.2 — Low dependency: the model is fairly robust to option ordering.
  • 0.2 ≤ ODS < 0.5 — Moderate dependency: some positional bias present.
  • ODS ≥ 0.5 — High dependency: strong positional bias; answers heavily influenced by ordering.
